# SRISRIGOLDTESTING
Sri Sri Gold Testing Shop - Official Website
This repository hosts the source code for the official website of Sri Sri Gold Testing Shop. The site serves as a digital storefront, providing customers with information on gold testing processes, live pricing updates, and shop location details.

Project Highlights

Responsive Design: Fully optimized for mobile and desktop viewing.

Service Showcase: Detailed sections on Fire Assay and XRF testing methods.

Live Rates: API integration to display current gold market rates.
